Landing Page + Website
Integration for Ralph Kerle's Art

Prepared by Doop UX  ·  ralphkerlesart.com

Context

The Problem to Solve

The landing page at commission.ralphkerlesart.com/homeinteriors and the main website at ralphkerlesart.com are currently operating as two isolated destinations with no navigation between them.

A visitor arriving at the commission page has no way to browse existing works before committing to an enquiry. A visitor browsing artworks on the main site sees nothing that says "if this doesn't fit your wall, I can make one that does." That missing connection is the primary conversion gap, and it needs to be resolved before the MailChimp campaign goes out.

Part 1 · Landing Page

homeinteriors.html Updates

What Needs Adding

  • Browse Available Works CTA: a mid-page button placed near the "How It Works" section, linking to the main artwork collection. This is the missing prompt. The footer already has links to individual collections and the official website, but there is no visible invitation earlier in the page for someone who is not ready to book yet.
Worth noting: ChatGPT suggested adding a "How the process begins" flow block, but the page already has a "How It Works" section covering exactly that. Adding another version of the same content would create redundancy without real benefit, so we left it as is.

Part 2 · Main Website

ralphkerlesart.com (Squarespace)

Navigation (Highest Priority)

  • Add "Commissions" to the main menu, linking directly to commission.ralphkerlesart.com/homeinteriors. This is the single most impactful change. Anyone on the site who sees Ralph's work and wants to know more has a clear path.

Individual Artwork Pages (High Priority)

  • Add a commission block below each artwork with the following copy: "Love this work but need a different size or format? I can create a commissioned piece based on this work, tailored specifically to your space."
  • CTA button: "Commission a Work" linking to commission.ralphkerlesart.com/homeinteriors
Heads up: this is the most complex part of the implementation. Squarespace doesn't have a consistent way to target individual artwork pages with JS since the DOM structure can vary by template and change with platform updates.

Collection Pages

  • Add a commission insert between artwork rows or at the top of each collection page: "Looking for something similar but tailored to your wall? Commission a Custom Work."

Homepage

  • One section only, not a hard sell: headline "From artwork to something entirely your own," with a "Learn About Commissions" CTA.

Announcement Bar (Campaign Launch Period)

  • Site-wide bar while the campaign is active: "Now accepting a limited number of commissioned works, book a consultation."

Footer

  • Add a Commissions column: Commission Overview, In-Home Consultation, Book a Call.
